Identify components that should be lazy loaded using AI.

## Install

```bash
npm install -g ai-lazy-load
```

## Usage

```bash
npx ai-lazy-load ./src/
npx ai-lazy-load ./src/components/
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
